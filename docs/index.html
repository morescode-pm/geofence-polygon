Kingdom Animalia Species Geofencing Tool
Instructions ▼

Draw a polygon on the map to discover animal species observed in that area:

  1. Search for your location using the search box
  2. Click the polygon tool () in the top-left corner to start drawing
  3. Click points on the map to create your polygon, double-click to finish
  4. Name your polygon and click "Save Geofence and Search" to find species

Species data provided by the Global Biodiversity Information Facility (GBIF) Occurrence API
